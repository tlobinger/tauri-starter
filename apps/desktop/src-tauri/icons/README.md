# Icons Directory

Place your application icons here:

- `32x32.png` - Small icon (32x32 pixels)
- `128x128.png` - Medium icon (128x128 pixels)
- `128x128@2x.png` - Retina medium icon (256x256 pixels)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these from a single source image using tools like:
- https://icon.kitchen/
- https://github.com/tauri-apps/tao (tauri icon command)

For now, the app will use default Tauri icons if these are not present.
